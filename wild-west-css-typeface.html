Wild West CSS Typeface

When creating a typeface made entirely out of css I decided to mimic old western inspired wood type specimens. However, because I was creating a typeface to live in a digital space I explored and pushed what a traditionally printed typeface meant when it lived on the web.

Because the form has a historical reference I chose a color palette that moved away from this to make it feel modern and dynamic, and appropriate for a web space. The animations accentuate the parts of the form that make it a classic wood typeface, which is a nod to where the form comes from. But, the animations themselves push this typeface into a digital realm, existing in a way that could only happen in this space.
